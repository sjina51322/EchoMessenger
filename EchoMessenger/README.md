\## 과제 내용



\- Label(표시), TextBox(입력), Button(전송), ListBox(대화창)를 배치한다.

\- 전송 버튼 클릭 시 TextBox의 텍스트를 ListBox의 항목(Items)에 추가한다.

\- 추가 직후 TextBox의 내용을 비워(Clear) 다음 입력을 준비한다.



\---



\## 구현 내용 및 기능 설명



\- 사용자가 입력창(TextBox)에 메시지를 입력하고 전송 버튼을 누르면 해당 메시지가 ListBox에 표시된다.

\- 메시지를 반복적으로 입력하면 ListBox에 한 줄씩 계속 추가된다.

\- 입력 후 TextBox는 자동으로 비워져 다음 입력을 바로 할 수 있다.

\- 메시지가 많아질 경우 ListBox에 스크롤바가 생성되어 내용을 확인할 수 있다.

\## 실행 화면 (과제2)



!\[과제2 실행화면](img/과제2.png) (img/과제2-1.png)



\---



\## 과제 내용



\- TextBox에서 엔터키 입력 시 메시지가 전송되도록 구현한다.

\- 입력 후 TextBox를 비워 다음 입력이 가능하도록 한다.

\- 입력창에 자동으로 포커스가 유지되도록 설정한다.



\---



\## 구현 내용 및 기능 설명



\- 사용자가 TextBox에 메시지를 입력한 후 엔터키를 누르면 버튼을 클릭하지 않아도 메시지가 ListBox에 추가된다.

\- 메시지가 추가된 후 TextBox의 내용은 자동으로 Clear() 되어 다음 입력을 준비한다.

\- 입력 후에도 커서가 TextBox에 유지되어 사용자가 계속해서 편리하게 입력할 수 있다.

\- 여러 메시지를 연속적으로 입력할 수 있어 사용성이 향상되었다.

\## 실행 화면 (과제3)



!\[과제3 실행화면](img/과제3.png)



\---



\## 과제 내용



\- 메시지 전송 시 현재 시간을 함께 표시한다.

\- ListBox에 저장된 메시지 개수를 화면에 표시한다.

\- 입력된 문자열의 앞뒤 공백을 제거하여 처리한다.



\---



\## 구현 내용 및 기능 설명



\- 사용자가 메시지를 입력하고 전송하면 현재 시간이 포함된 형태로 ListBox에 표시된다. (예: \[12:38:37] 안녕)

\- 메시지가 추가될 때마다 전체 메시지 개수를 계산하여 Label을 통해 "현재 대화: n개" 형태로 출력한다.

\- 입력된 문자열은 Trim()을 사용하여 앞뒤 공백을 제거한 후 처리되도록 구현하였다.

\- 이를 통해 보다 깔끔한 메시지 출력과 정보 확인이 가능하도록 하였다.
